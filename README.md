# 📊 Tekvi Sprint Dashboard

Agile Scrum board cho Tekvi team. 

👉 **Xem online**: https://ithhlvn.github.io/tekvi-sprint-dashboard/

## Tính năng
- Board 4 cột: To Do / In Progress / Done / Blocked
- Thống kê tiến độ sprint
- Thêm task trực tiếp trên giao diện
- Daily reports log
- Export/Import JSON

## Cách dùng
1. Mở link trên
2. Thêm task bằng form bên dưới board
3. Dùng Export JSON để backup
4. Mở lại: dữ liệu lưu trên trình duyệt (localStorage)
