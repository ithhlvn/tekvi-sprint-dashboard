> 📍 Gửi vào thread Daily trong group Tekvi Team

## Format chuẩn (copy-paste)

```
🔹 Tên: [Tên chuyên gia / bot]
✅ Sprint #:
🗓️ Ngày:

✅ Đã làm:
- [ ] Task 1: ...
- [ ] Task 2: ...

📋 Sẽ làm (hôm nay):
- [ ] Task 1: ...
- [ ] Task 2: ...

⚠️ Khó khăn / Blocking:
- [ ] ...

💡 Giải pháp / Đề xuất:
- ...
```

## Ví dụ

```
🔹 Tên: Bảo (Backend)
✅ Sprint #: 1
🗓️ Ngày: 01/07/2026

✅ Đã làm:
- [x] Thiết kế DB user module
- [x] Viết API login + register

📋 Sẽ làm (hôm nay):
- [ ] Middleware JWT
- [ ] API CRUD user profile

⚠️ Khó khăn / Blocking:
- [ ] Chờ Tú giao UI mẫu login mới deploy thử

💡 Giải pháp / Đề xuất:
- Em làm middleware xong trước, chạy unit test trước, khi có UI ghép sau.
```

## Lưu ý

- ✅ **Task đã xong**: ghi rõ nội dung, dùng `[x]`
- 📋 **Sẽ làm**: ưu tiên theo sprint backlog
- ⚠️ **Khó khăn**: không ngại nêu blocking, càng sớm càng tốt
- 💡 **Giải pháp**: tự đề xuất trước khi chờ người khác
- **Tag người hỗ trợ** khi cần: `@username`
